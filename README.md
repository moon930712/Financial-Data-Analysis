# Financial Data Analysis

금융 데이터를 수집하고 개인적인 퀀트 분석 모델을 설계 및 테스트하기 위해 작성 중인 프로젝트 기록입니다.

## 목적
- PostgreSQL과 파이썬을 연동한 정규 데이터 파이프라인 구축.
- 과거 시계열 데이터를 활용해 업종별 상대 평가와 기업 펀더멘탈을 추적하는 모델 구현.

## 주요 스크립트 및 내용
- DB 시스템 카탈로그 파싱을 통한 스키마 메타데이터 추출 자동화.
- 개별 기업의 3년 평균 시계열 데이터(PBR, PER 등)를 바탕으로 한 가치 평가 С크리닝.
- 펀더멘탈 지표, 저평가 밀집도, 수급 모멘텀 등을 결합한 업종별 Z-Score 모델 설계.

새로운 계량 투자 전략이나 데이터 파이프라인 최적화를 공부하면서 주기적으로 스크립트를 추가하고 기록할 예정.
