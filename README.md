# Financial Data Analysis (포트폴리오)

이 프로젝트는 금융 데이터베이스의 스키마 구조를 분석하고, 실무 수준의 복잡한 금융 데이터를 체계적으로 추출 및 파이프라인화하기 위해 만들어진 데이터 분석 프로젝트입니다.

## 🌟 프로젝트 개요
- **목적:** 데이터베이스 스키마 추출, 전처리 자동화, 그리고 분석용 데이터 파이프라인 구축
- **기술 스택:** Python, Pandas, PostgreSQL, Git

## 💡 주요 기능
1. **스키마 및 테이블 메타데이터 자동 추출**
   - 수백 개의 테이블과 컬럼 설명을 데이터베이스 System Catalog에서 동적으로 파싱
   - `.xlsx` 및 `.md` 형식으로 구조화하여 자동 문서화
2. **보안 및 환경 변수 관리**
   - 인증 정보와 로컬 환경을 분리하여 안전한 코드 관리 유지

*(채용 담당자를 위해 향후 주요 데이터 분석 결과물과 시각화, 아키텍처 다이어그램 등을 이 곳에 꾸준히 추가할 예정입니다.)*
